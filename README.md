# Slack-voting
A voting extension for Slack developed by FrontendDevelopers.org

To view current poll information:
``
/poll
``

To vote on current poll:
``
/vote [Y / N]
``

To view all regions:
``
/region list
``

To join a region:
``
/region reg_[loc]
``
``
/region reg_uk
``

To create a region:
``
/region create [loc]
``

``
/region create uk
``
